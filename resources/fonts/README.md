# Fira Sans Font Files

This directory should contain the Fira Sans font family files.

## Required Font Files

Download Fira Sans from Google Fonts or Mozilla:
- https://fonts.google.com/specimen/Fira+Sans
- https://github.com/mozilla/Fira

Place the following font files in this directory:

### Required variants:
- `FiraSans-Regular.ttf` - Regular weight
- `FiraSans-Medium.ttf` - Medium weight (500)
- `FiraSans-SemiBold.ttf` - Semi-bold weight (600)
- `FiraSans-Bold.ttf` - Bold weight (700)

### Optional (for complete family):
- `FiraSans-Light.ttf` - Light weight (300)
- `FiraSans-ExtraBold.ttf` - Extra bold weight (800)
- `FiraSans-Italic.ttf` - Italic variants

## Installation

1. Download the Fira Sans font family from one of the sources above
2. Extract the `.ttf` files
3. Copy at minimum the 4 required variants listed above to this directory
4. Restart the application

The application will automatically load any `.ttf` or `.otf` font files found in this directory.
